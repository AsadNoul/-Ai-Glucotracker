# GlucoTrack AI - Icon Design Specifications

## Main App Icon Design

### Concept
A modern, minimalist blood drop with a glucose monitoring theme that works perfectly in both dark and light modes.

### Design Elements
1. **Primary Shape**: Rounded blood drop (teardrop shape)
2. **Color Scheme**: 
   - Main gradient: #0a85ff → #137fec (vibrant cyan-blue)
   - Background: #0f1923 (dark navy) with subtle radial gradient
   - Accent: White/light blue highlights for glass morphism effect

3. **Inner Detail**: 
   - Option A: Subtle EKG heartbeat line
   - Option B: Simple glucose curve graph
   - Option C: Tiny plus/medical cross symbol

4. **Style**:
   - Glass morphism/liquid glass aesthetic
   - Soft shadows and highlights
   - Smooth gradients
   - Professional medical feel

### Icon Specifications

#### App Icon (icon.png)
- **Size**: 1024x1024px
- **Format**: PNG with transparency
- **Corner Radius**: Applied by system
- **Safe Area**: Keep important elements within 90% of canvas

#### Adaptive Icon (Android)
- **Foreground**: 1024x1024px (icon with padding)
- **Background**: Solid color #0f1923 or gradient
- **Safe Area**: 66% circle in center

#### Splash Screen
- **Size**: 1284x2778px (tallest iPhone)
- **Design**: Same icon centered on gradient background
- **Background**: Animated gradient #0f1923 → #101922
